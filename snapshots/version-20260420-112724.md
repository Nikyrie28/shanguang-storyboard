# 闪光分镜版本说明

版本时间：`2026-04-20 11:27:24`

对应快照文件：
`snapshots/index-20260420-112724.html`

## 本次版本已更新内容

1. 分栏拖拽逻辑已调整为“拖哪一栏就只改哪一栏”，并限制最左和最右边界不可拖动。
2. `复制` 列固定贴在最右侧，空余空间会自动补满，避免表头右侧出现白色空缺。
3. `运镜` 左右两侧分隔支持调整宽度。
4. 图片删除按钮改为稳定的固定定位，不再出现悬浮偏移。
5. 新增 `列设置` 面板，支持：
   - 自定义每一列表头名称
   - 自定义表头图标
   - 隐藏 / 恢复已有列
6. 图标选择器已改为可视化面板，不再使用纯文字下拉。
7. 图标库已扩充，并新增了 `人物` 图标，可用于 `拍摄对象` 等列。
8. 原 `部门` 列默认文案已改为 `拍摄对象`：
   - 表头默认值为 `拍摄对象`
   - 单元格默认提示为 `点击输入拍摄对象`
   - 旧项目里残留的 `部门 / 点击输入部门` 也会自动兼容迁正
9. 批量模式已升级：
   - 支持整行勾选
   - 支持单元格多选
   - 支持底部悬浮操作栏
   - 支持批量删除、批量复制行、批量清空
10. 批量选择交互已支持接近系统行为的逻辑：
   - 普通点击单选
   - `Ctrl / Cmd` 多选
   - `Shift` 连续范围选择
11. 批量输入已改为实时输入模式：
   - 点击 `批量输入` 后直接键入
   - 所有选中单元格会同步变成同样的内容
12. 批量粘贴已改为统一覆盖模式：
   - 粘贴内容会直接覆盖所有选中单元格
   - 不再按矩阵区域铺贴
13. 当使用“整行勾选”时，`批量输入` 和 `批量粘贴` 会自动变灰并禁用。
14. 时间规则已修正：
   - `结束时间` 始终大于等于 `开始时间`
   - 若开始时间更大，则自动把结束时间追平到开始时间
15. 项目顶部 `拍摄日期` 已改成滚轮选择器：
   - 使用 `年 / 月 / 日` 三列滚轮
   - 点击字段弹出
   - 确认后写回 `YYYY-MM-DD`

## 当前版本已有能力

1. 项目创建、编辑、删除、进入。
2. 项目基础信息编辑：
   - 项目名称
   - 拍摄日期
   - 导演
   - 拍摄地点
   - 总时长
3. 分镜行新增、插入、删除、复制、拖拽排序。
4. 时间滚轮选择与上下镜头时间联动。
5. 参考图上传、显示、删除。
6. 表头列宽拖拽。
7. 列显示隐藏与列标题/图标自定义。
8. 批量模式下的行级与单元格级操作。
9. 导入 / 导出能力：
   - JSON 导入导出
   - CSV 导入
   - HTML 导出
   - PDF 导出
10. 项目内容保存与当前列设置持久化。
