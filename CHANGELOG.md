# 更新日志 (Changelog)

## [2026-04-21] 最新更新

### ✨ 优化与改进 (Enhancements)
- **悬浮选择模式**：彻底重构了此前的“批量模式”，现在点击“选择”按钮后会在页面底部弹出丝滑的毛玻璃悬浮操作条。支持行与单元格选择的严格**互斥逻辑**，大幅降低误操作概率。
- **图片自动压缩**：导入分镜参考图时将自动进行本地压缩处理（限制最长边并适当降低质量），大幅减少了 Base64 编码体积与服务器存储压力。
- **CSV 导入逻辑调整**：由于图片处理机制更新，目前导入 CSV 数据时将自动忽略参考图的解析与导入，保持数据的纯净性。
- **动态自适应行高**：移除了“运镜”列的固定宽度限制，现支持根据屏幕与分镜比例变化动态自适应单元格宽度及文字换行，实现了行高的完美联动。
- **页面布局适配**：移除了首页中间主内容区的最大宽度限制，将其调整为 `w-full`。在宽屏显示器下，白色工作区能够向两侧完全展开并填满页面，提升了屏幕空间利用率与视觉整体感。
- **UI 风格统一**：将主页左侧菜单栏顶部的闪电 Emoji（⚡）替换为了基于项目内建图标库（Heroicons）的 SVG 线框图标，统一了全站的图标视觉规范。
- **全新导入交互体验**：重构了文件导入流程。新增了与 CSV 导入高度一致的 JSON 导入可视化弹窗，支持文件拖拽。且在所有导入面板中新增了“文档就绪图标”和文件移除按钮（`×`），使用户在确认导入前可以直观预览文件状态并支持便捷重新上传。

### 🐛 问题修复 (Bug Fixes)
- **表头圆角溢出修复**：为包含整个表格的外部卡片增加了 `overflow-hidden` 属性，彻底修复了顶部表头灰色背景超出外层圆角边界导致出现“锐角”的视觉 Bug。
- **Logo 状态异常**：修复了在进入“项目视图（Project View）”时，左上角品牌 Logo（`logo-frame-2.png`）被意外隐藏的代码逻辑。现在 Logo 会在顶部导航栏正确常驻显示。
